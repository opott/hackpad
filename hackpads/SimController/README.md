# SimController

This is a 15-key macropad designed for simulation gamers.

The number of keys is ideal for people playing games such as Train Sim World, OMSI 2 or Microsoft Flight Simulator.

Initial firmware was hard to make as I obviously did not have the macropad in front of me. It was created using POG.
The firmware will hopefully be improved over time to create different 'profiles' made specifically for each game.


## BOM
- PCB
- 2-part Case
- Seeed XIAO RP2040 Microcontroller
- 15 Switches
- 15 1N4148 Diodes
